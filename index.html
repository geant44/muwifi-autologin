MU-WiFi Autologin for Android

An android application to automatically login to MU-WiFi.
แอพฯแอนดรอยด์สำหรับล็อกอินเข้าสู่ระบบ MU-WiFi อัตโนมัติ

Download ดาวน์โหลด

Download in Android Market.

(support Android 2.1+)

Usage วิธีใช้

1. Set the correct username, password and tick Enabled.
2. Connect to the MU-WiFi network and the application will automatically login for you.

Remark: The "Login Now!" option is used to force the login procedure to occur. It is normally not needed.

1. ตั้งค่า username และ password ให้ถูกต้อง หลังจากนั้นให้ติ๊กช่อง Enabled
2. เชื่อมต่อเข้าสู่ระบบ MU-WiFi และแอพฯจะทำการล็อกอินเข้าสู่ระบบอัตโนมัติ

หมายเหตุ: ตัวเลือก "Login Now!" มีไว้สำหรับให้แอพฯทำการล็อกอินเข้าสู่ระบบทันที โดยปกติแล้วจะไม่ต้องใช้งาน

The display for a successful login

การแสดงผลเมื่อการล็อนอินสำเร็จ

The display for when a login is not required.

การแสดงผลเมื่อไม่ต้องมีการล็อกอิน

Troubleshooting การแก้ไขข้อผิดพลาด

When an error has occurred, it will be put in the notification bar. For certain kinds of errors, you can click the item in the notification bar to view the detailed error message.

เมื่อเกิดข้อผิดพลาด แอพฯจะแสดงผลในแถบแจ้งเตือนและสำหรับข้อผิดพลาดบางชนิดจะสามารถดูรายละเอียดของข้อผิดพลาดได้โดยการกดรายการของแอพฯในแถมแจ้งเตือน
